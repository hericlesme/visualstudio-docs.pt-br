# <a name="install-a-uml-profile"></a>Instalar um perfil UML
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [instalar um perfil UML](https://docs.microsoft.com/visualstudio/modeling/install-a-uml-profile).  
  
Você pode estender o Visual Studio com um perfil UML. Um perfil permite adicionar estereótipos e propriedades adicionais para os elementos que você pode criar em modelos UML. Para ver quais versões do Visual Studio dão suporte a esse recurso, consulte [suporte de versão para a arquitetura e ferramentas de modelagem](../modeling/what-s-new-for-design-in-visual-studio.md#VersionSupport).  
  
 Se você receber um modelo UML que foi criado usando perfis, algumas propriedades não serão exibidas a menos que você instale os mesmos perfis.  
  
 Um perfil é distribuído dentro de uma extensão do Visual Studio. Uma extensão também pode conter outros recursos, como comandos de menu. Para obter mais informações, consulte [gerenciamento de extensões do Visual Studio](http://go.microsoft.com/fwlink/?LinkId=160728).  
  
### <a name="to-install-a-uml-profile-on-your-computer"></a>Para instalar um perfil UML em seu computador  
  
1.  O perfil deve ter sido fornecido a você na forma de uma extensão do Visual Studio (`.vsix`) arquivos. Pode haver outros recursos no mesmo arquivo.  
  
     Mover o `.vsix` arquivo em um local conveniente em seu computador.  
  
2.  Clique duas vezes o `.vsix` no Windows Explorer (ou Explorador de arquivos) de arquivos ou abri-lo em [!INCLUDE[vs_current_short](../includes/vs-current-short-md.md)].  
  
3.  Clique em **instalar** na caixa de diálogo que aparece.  
  
4.  Para desinstalar ou desabilitar temporariamente a extensão, abra **Gerenciador de extensões** da **ferramentas** menu.  
  
### <a name="to-uninstall-or-disable-a-profile-extension"></a>Para desinstalar ou desabilitar uma extensão de perfil  
  
1.  No Visual Studio **ferramentas** menu, clique em **Gerenciador de extensões**.  
  
2.  Clique na extensão que você deseja remover e, em seguida, clique em **desabilite** ou **desinstalar**.
