# <a name="reference-visual-studio-debugging-apis"></a>Referência (APIs de depuração no Visual Studio)
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [referência (Visual Studio depurando APIs)](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/reference-visual-studio-debugging-apis).  
  
As referências de SDK de depuração do Visual Studio descrevem o modelo básico para a arquitetura de tempo de execução e a funcionalidade do depurador e seus componentes. A documentação faz suposições o menor número possível sobre os componentes e interfaces que ele descreve, além do comportamento define por meio de interfaces.  
  
 O objetivo da seção de referência é fornecer suporte para como diversificado um conjunto de linguagens, tempos de execução e sistemas operacionais possíveis.
